# WeChat-app-sample-code

微信小程序示例代码

## 项目介绍

微信小程序开发过程中，遇到的一些问题，或者解决方法，示例代码记录

## 示例项目（此项目）基于技术栈

> react 17 + TS 4.x + Taro 3.x + Taro UI 3.x + dva + less

直接是基于 GitHub 上[wechat-app](https://github.com/kivet-h/wechat-app) 这个项目拉的一个

## 环境要求

> node >=12.20.0

## 运行

```bash
$ yarn start

$ npm run start
```

## Git 规范

```bash
# 提交代码
$ yarn commit
```

## 构建

```bash
$ yarn build
```

## 查看 taro 版本

```bash
$ taro info
```
